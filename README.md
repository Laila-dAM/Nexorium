## Overview

Nexorium is a modern fullstack ERP SaaS platform designed to centralize business operations into a single intelligent system.

The platform combines:
- enterprise architecture
- real-time dashboards
- analytics
- productivity tools
- scalable backend infrastructure
- modern futuristic UI/UX

Built to simulate a real-world commercial SaaS application.

# Installation

## Frontend

```bash
cd frontend
npm install
npm run dev
```

Frontend server:
```bash
http://localhost:5173
```

---

## Backend

```bash
cd backend
npm install
npm run dev
```

Backend server:
```bash
http://localhost:3333
```

---

# Environment Variables

## Backend

Create a `.env` file inside `backend/`:

```env
DATABASE_URL=
JWT_SECRET=
```
